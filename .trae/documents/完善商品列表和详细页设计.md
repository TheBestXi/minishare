# 完善商品列表和详细页设计方案

## 1. 设计目标

- 改进商品列表页，提升视觉效果和用户体验
- 重新设计商品详细页，包含：
  - 商品图片轮播（支持左右按钮切换）
  - 清晰的价格展示
  - 详细的商品介绍
  - 评论区功能

## 2. 前端代码设计

### 2.1 商品列表页改进（Products/Index.cshtml）

- 采用卡片式布局，展示商品图片、名称、价格和简介
- 添加悬停效果，提升交互体验
- 优化响应式设计，适配不同屏幕尺寸

### 2.2 商品详细页重新设计（Products/Details.cshtml）

- 左侧：商品图片轮播区
  - 主图展示
  - 缩略图导航
  - 左右切换按钮
- 右侧：商品信息区
  - 商品名称
  - 价格展示
  - 商品描述
  - 加入购物车按钮
- 下方：评论区
  - 评论列表
  - 评论表单

### 2.3 相关CSS样式

- 添加商品卡片样式
- 设计图片轮播样式
- 优化评论区样式
- 确保响应式设计

## 3. 实现步骤

### 3.1 前端实现

1. 修改 `Views/Products/Index.cshtml`，改进商品列表展示
2. 重新设计 `Views/Products/Details.cshtml`，添加图片轮播和评论区
3. 在 `wwwroot/css/site.css` 中添加相关样式
4. 在 `wwwroot/js/site.js` 中添加图片轮播和评论交互逻辑

### 3.2 控制器扩展（可选）

如果需要创建新的控制器类：

1. 在 `Controllers` 目录下创建新的控制器类（如 `ProductDetailsController.cs`）
2. 继承 `Controller` 类
3. 添加必要的依赖注入（如 `ApplicationDbContext`）
4. 实现相关动作方法（如 `Details`、`GetComments`、`AddComment` 等）
5. 在 `Startup.cs` 中配置路由（如果需要）
6. 创建对应的视图文件

## 4. 技术要点

- 使用 Bootstrap 5 组件构建布局
- 使用 jQuery 实现图片轮播交互
- 采用响应式设计，适配不同屏幕尺寸
- 遵循 MVC 设计模式，分离视图和控制器逻辑

## 5. 预期效果

- 商品列表页更加美观，交互体验更好
- 商品详细页包含完整的商品信息和图片展示
- 支持商品图片轮播查看
- 包含评论功能，增强用户互动
